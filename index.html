The Crystal Game

How the game works...

The computer will display a number in the green box below.

There are four crystals that you can select from. Each crystal has a randomly assigned number between 1 and 12. The number associated with each crystal can only be determined as they are selected, so you have to remember each crystal's value as you proceed through the game.

The object of the game is to choose a sequence of crystals that match the computer generated "target number."
